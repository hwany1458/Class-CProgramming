# [Class] C언어 프로그래밍

원광대학교 공학 3계열 1학년 **C언어프로그래밍** 강의용 GitHub 저장소입니다.  
본 저장소는 C 언어의 기초 문법부터 문제 해결을 위한 구조적 프로그래밍 방법까지 학습할 수 있도록, 강의자료·예제코드·실습자료·과제 참고자료를 정리한 수업 지원용 레포지토리입니다.

이 강의는 C 언어를 통해 컴퓨터 프로그래밍의 기본 문법과 동작 구조를 익히고, 주어진 문제를 프로그램으로 구현하는 능력을 기르는 것을 목표로 합니다. 또한 Visual Studio 통합 개발환경을 활용하여 직접 코드를 작성하고 실행하면서 프로그래밍의 실제 흐름을 익히도록 구성되어 있습니다.

## 🏆 Course Goals

본 교과목의 주요 목표는 다음과 같습니다.

1. C 언어의 기본 개념과 문법을 이해한다.
2. 데이터형, 연산자, 제어문, 함수, 배열, 문자열 등의 핵심 개념을 익힌다.

4. 주어진 문제를 C 언어로 구조적으로 분석하고 해결할 수 있다.
5. 알고리즘적 사고를 바탕으로 문제 해결형 프로그래밍 능력을 기른다.

## 🔅 Course Philosophy

C 언어는 모든 프로그래밍의 기초를 이루는 중요한 언어입니다.
이 강의에서는 단순히 C 문법만 배우는 것이 아니라,

* 프로그램이 어떻게 구성되는지,
* 데이터가 어떻게 저장되고 처리되는지,
* 조건과 반복으로 문제를 어떻게 해결하는지,
* 함수와 배열을 사용해 프로그램을 어떻게 구조화하는지

를 체계적으로 학습하는 것을 목표로 합니다.

따라서 이 저장소 역시 단순 예제 나열이 아니라,
**기초 문법 → 문제 해결 → 응용 확장**의 흐름에 맞춰 구성하는 것을 지향합니다.
